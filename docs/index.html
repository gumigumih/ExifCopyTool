できること

右クリックでコピー

ファイルを選んでフォーマットを選ぶだけ。常駐せず、必要な時だけ起動します。

フォーマット編集

{Make} や {LensModel} などのタグを組み合わせて、コピー内容を自由に作れます。

RAWにも対応

JPEGやPNGに加えて、ARW、CR3、NEFなどから取得できるEXIF情報を読み取ります。

使い方

  1. インストーラーをダウンロードして実行します。
  2. 設定画面で右クリックメニューを有効にします。
  3. 写真ファイルを右クリックし、「EXIF情報をコピー」からフォーマットを選びます。
  4. 整形されたEXIF情報がクリップボードへコピーされます。

配布情報

インストール先は %LOCALAPPDATA%\Programs\ExifCopyTool です。Pythonは不要です。アンインストール時に右クリックメニューも解除します。
